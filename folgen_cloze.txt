Besitzt eine Folge keinen Grenzwert, heißt sie {{c1::unbestimmt divergent}}.;folgen
Besitzt eine Folge einen eindeutigen Grenzwert, ist sie {{c1::bestimmt konvergent}}. Besitzt sie den Grenzwert {{c2::+/- unendlich} heißt sie unbestimmt konvergent bzw. bestimmt divergent.;folgen
Der größte Häufungspunkt heißt: {{c1::Limes superior}}.;folgen
Der kleinste Häufungspunkt heißt: {{c1::Limes inferior}}.;folgen
Eine Folge heißt monoton fallend, wenn {{c1::xn+1 <= xn}}.;folgen
Eine Folge heißt streng monoton fallend, wenn {{c1::xn+1 < xn}}.;folgen
Eine Folge heißt monoton steigend, wenn {{c1::xn+1 >= xn}}.;folgen
Eine Folge heißt streng monoton steigend, wenn {{c1::xn+1 > xn}}.;folgen
Jede {{c1::konvergente Folge}} ist {{c2::beschränkt}}.;folgen
Jede {{c1::beschränkte Folge}} besitzt mindestens einen {{c2::Häufungswert}} (Satz von {{c3::Bolzano-Weierstrass}}).;folgen
Eine {{c1::monotone Folge}} ist genau dann {{c2::konvergent}}, wenn sie {{c2::beschränkt} ist. ({{c3::Hauptsatz über monotone Folgen}});folgen
Jede nach {{c1::oben beschränkte Folge}} besitzt ein {{c2::Supremum}}.;folgen
Jede nach {{c1::unten beschränkte Folge}} besitzt ein {{c2::Infimum}}.;folgen
